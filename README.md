## Kompilowanie

Wykonaj te komendy w folderze z plikami modyfikacji, jeśli chcesz samodzielnie kompilować modyfikację:

```shell
./gradlew build
./gradlew clean build
```
Plik powinien znajdować się w /build/libs/
